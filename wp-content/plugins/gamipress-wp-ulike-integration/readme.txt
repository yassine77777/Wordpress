Connect GamiPress with WP Ulike

== Description ==

Gamify your [WP Ulike](http://wordpress.org/plugins/wp-ulike/ "WP Ulike") interactions thanks to the powerful gamification plugin, [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress")!

This plugin automatically connects GamiPress with WP Ulike adding new activity events.

= New Events =

* Like anything: When a user likes anything.
* Like a post: When a user likes a post.
* Like a post of a type: When a user likes a post of a type.
* Like a comment: When a user likes a comment.
* Get a like anywhere: When an author gets a like anywhere.
* Get a like on a post: When an author gets a like on a post.
* Get a like on a post of a type: When an author gets a like on a post of a type.
* Get a like on a comment: When an author gets a like on a comment.
* Unlike anything: When an user unlikes anything.
* Unlike a post: When an user unlikes a post.
* Unlike a comment: When an user unlikes a comment.
* Get an unlike anywhere: When an author gets an unlike anywhere.
* Get an unlike on a post: When an author gets an unlike on a post.
* Get an unlike on a comment: When an author gets an unlike on a comment.

= BuddyPress Events =
* Like an activity: When an user likes an activity.
* Get a like on an activity: When an author gets a like on an activity.
* Unlike an activity: When an user unlikes an activity.
* Get an unlike on an activity: When an author gets an unlike on an activity.

= bbPress Events =
* Like a topic: When an user likes a topic.
* Get a like on a topic: When an author gets a like on a topic.
* Unlike a topic: When an user unlikes a topic.
* Get an unlike on a topic: When an author gets an unlike on a topic.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
